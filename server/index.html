Compteur des API

  • GET /api/config

    Récupérer la configuration

  • POST /api/config

    Mettre à jour la configuration

  • GET /api/jets

    Récupérer les jets

  • POST /api/jets

    Ajouter un nouveau jet

  • DELETE /api/jets

    Supprimer tous les jets

    Récupérer les séances

  • POST /api/seances

    Ajouter une nouvelle séance

  • PUT /api/seances/:id

    Modifier une séance existante
